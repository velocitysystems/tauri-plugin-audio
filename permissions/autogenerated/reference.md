## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-load`
- `allow-play`
- `allow-pause`
- `allow-stop`
- `allow-seek`
- `allow-set-volume`
- `allow-set-muted`
- `allow-set-playback-rate`
- `allow-set-loop`
- `allow-get-state`
- `allow-is-native`
